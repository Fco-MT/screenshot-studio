# 📱 App Store Screenshot Studio

A free, offline-capable tool for creating polished App Store screenshots — right in your browser. No installation, no account, no cloud uploads. Just open the HTML file and start designing.

## ✨ Features

**Device Presets**
iPhone 6.9" (1320×2868), iPhone 6.7" (1290×2796), iPad 13" (2064×2752), iPad 12.9" (2048×2732) — portrait and landscape.

**Backgrounds**
Two- or three-color gradients with direction control, solid colors, or custom background images with blur and darken effects. Includes 10 ready-made gradient presets.

**Device Frames**
Realistic dark or light device shells with Dynamic Island for iPhones. Adjustable size from 45% to 95%.

**Text Overlays**
Headline and subtitle with independent font sizes, adjustable gap, free X/Y positioning, and manual line breaks via Enter. Four built-in fonts: Nunito, Poppins, Playfair Display, and Noto Sans TC. Italic toggle included.

**Per-Image Settings**
Each screenshot has its own headline, subtitle, text position, image position, and tilt angle (-30° to +30°). One-click copy to apply settings across all images.

**Batch Workflow**
Upload up to 10 screenshots at once (click or drag & drop), navigate with thumbnails or arrow keys, and export individually, all at once, or in all four device sizes with a single click.

**Apple-Ready Export**
All exports are opaque RGB PNGs — no alpha channel, no App Store Connect rejections.

**Trilingual Interface**
Full UI in English, German (Deutsch), and Traditional Chinese (繁體中文), switchable with one click.

**100% Offline**
All fonts are embedded as Base64. The entire tool is a single 1 MB HTML file with zero external dependencies.

## 🚀 Getting Started

1. Open the [live version](https://Fco-MT.github.io/screenshot-studio/) — or download `index.html` and open it locally
2. Select your target device
3. Upload your app screenshots
4. Customize backgrounds, text, and positioning per image
5. Export — done!

## ⌨️ Keyboard Shortcuts

| Shortcut | Action |
|---|---|
| `←` `→` | Previous / next image |
| `Ctrl/⌘` + `+` | Zoom in |
| `Ctrl/⌘` + `-` | Zoom out |
| `Ctrl/⌘` + `0` | Reset zoom to 100% |
| `Ctrl/⌘` + Scroll | Zoom with mouse wheel |

## 💡 Tips

- **Xcode Simulator screenshots** work perfectly as input — just press `⌘ S` in the Simulator
- Use the **"Apply to all"** button to quickly set uniform text across all screenshots, then customize individual headlines
- For **Chinese text**, select Noto Sans TC as the overlay font for best rendering
- The **Fit** button in the zoom bar instantly adjusts the preview to your screen size
